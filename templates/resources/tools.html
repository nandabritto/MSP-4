Flying Donut

Flying Donut is a powerful, simple, effective, collaborative online scrum and kanban software for public and private projects.

The easy to use backlog buckets provide an easier way to manage the backlog. Use it to keep your unscheduled work and move the cards to your active kanban or scrum board when its time to start working on them.

JSHint

JSHint is a program that flags suspicious usage in programs written in JavaScript. The core project consists of a library itself as well as a CLI program distributed as a Node module.

JSHint comes with a default set of warnings but it was designed to be very configurable.

WAI Color Assessor

The Web Accessibility Institute Color Assessor is a tool that compares the #HEX details of background and Foreground colors to ensure your project falls within WAI standards.

Success Criterion 1.4.3 of WCAG 2.0 requires the visual presentation of text and images of text has a contrast ratio of at least 4.5:1.

There are three exceptions to this standard which are detailed on this site.

Learn More...
WAVE Accessibility

WAVE is a suite of evaluation tools that helps authors make their web content more accessible to individuals with disabilities. WAVE can identify many accessibility and Web Content Accessibility Guideline (WCAG) errors, but also facilitates human evaluation of web content.

Their philosophy is to focus on issues that we know impact end users, facilitate human evaluation, and to educate about web accessibility.

Learn More...
tinypng.com

TinyPNG uses smart lossy compression techniques to reduce the file size of your WEBP, JPEG and PNG files.

By selectively decreasing the number of colors in the image, fewer bytes are required to store the data. The effect is nearly invisible but it makes a very large difference in file size!
