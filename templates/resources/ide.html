IDE? What on earth is that?!?!

IDE stands for Integrated Development Environment. This software is essentially a collection of developer tools squashed into a nice neat graphical user interface (GUI)

There are numerous IDEs available, everything from a simple notepad type program to a fully loaded system such as VS Code.

Each IDE has a primary function and has its own list of pros and cons. Some are built with a focus towards a specific development language, while others support multiple languages via various plug ins and add ons
